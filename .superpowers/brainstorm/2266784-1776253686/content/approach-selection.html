TinkerClaw Mode: What Should It Feel Like?

TinkerClaw is a full agent (51 skills, memory, web search, tools) but the Tab5 currently treats it as just another chatbot. How should the UI reflect its capabilities?

A

Enhanced Chat

Same chat screen, but smarter. Agent activity indicators during tool execution ("Searching the web..."), memory-aware suggestions, tool result cards inline. The conversation IS the interface — you ask via voice/text, it reports back in the chat stream.

Think: iMessage, but the other person is an AI agent that can do things
Effort: Medium — enhances existing ui_chat.c
LVGL objects: +5-8 objects (fits within 55 limit)
Risk: Low — no new screens, no new navigation
B

Agent Dashboard

A brand new dedicated screen with zones: conversation panel on the left, tool activity feed, memory viewer, skill quick-launch grid. The agent's internal state is always visible — you can see what it knows, what tools it has, what it's doing.

Think: Mission control for your AI agent
Effort: High — new ui_agent.c (~1500 lines), new screen in tileview
LVGL objects: +25-35 objects (pushes the 55 limit hard)
Risk: High — 720px wide screen is too narrow for panels, heap pressure
C

Mode Morphing

No new screen — existing screens transform when TinkerClaw is active. Home shows agent status + recent activity instead of just a clock. Chat gets tool indicators + agent suggestions. Settings shows agent config + capabilities. The whole device feels different in agent mode.

Think: The device shapeshifts based on what mode you're in
Effort: Medium-High — changes across 3 UI files
LVGL objects: +8-12 objects (reuses existing, swaps content)
Risk: Medium — conditional code in multiple files, testing across all modes

Context: Device Constraints

  • 720x1280 display — portrait only, no room for side panels
  • ~55 LVGL object limit — internal SRAM fragments with more
  • Voice-first — most interactions are spoken, not tapped
  • 32MB PSRAM — plenty for data, but LVGL objects use internal SRAM
  • Current screens: Home, Chat, Notes, Settings, Camera, Files (all working)
