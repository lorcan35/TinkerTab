Rich Media Chat: 120 User Stories

30 stories per category — real scenarios for an AI voice/chat device

A: AI Generates / Finds Media

The AI's response contains images, audio, video, or web content that gets rendered inline

A1. Recipe with photo

"How do I make carbonara?" — AI responds with steps AND a photo of the finished dish from web search. Image renders inline in the chat bubble.

A2. Weather with map

"What's the weather in Dublin?" — AI responds with forecast text + a weather radar image or icon showing conditions.

A3. Product comparison

"Compare AirPods Pro vs Sony XM5" — AI responds with text comparison + product images side by side.

A4. Place lookup

"Find Italian restaurants near me" — AI responds with names/ratings + a map thumbnail or photos of the restaurants.

A5. News article with image

"What happened in tech today?" — AI summarizes headlines with thumbnail images from articles. Tap to open full article preview.

"How do I tie a bowline knot?" — AI responds with step-by-step text + an instructional diagram/image.

A9. Movie recommendation with trailer

"What should I watch tonight?" — AI suggests a movie with poster image + link to trailer that plays inline.

A10. Math with rendered equation

"Explain the quadratic formula" — AI responds with explanation + beautifully rendered LaTeX equation as an image.

A11. Travel destination

"Tell me about Kyoto" — AI responds with history/tips + beautiful photos of temples, streets, food.

"How do you pronounce 'croissant' in French?" — AI responds with text + a short audio clip of the pronunciation.

D: Cross-Cutting / Multi-Modal Flows

Stories that combine multiple media types or span input→processing→output in interesting ways

D1. Photo → AI image edit → result

User sends a photo → "Remove the background" → TinkerClaw processes → returns edited image inline.

D2. Voice → search → images + audio

User asks "What does a theremin sound like?" → AI searches → responds with description, photo of instrument, AND audio sample.

D3. Photo → chart → voice explanation

User photographs a spreadsheet → Dragon OCRs → creates chart → explains trends via voice + shows chart image.

D4. Conversation → save as document

"Save this conversation as a PDF" → Dragon renders the full chat (text + images + code) as a styled PDF → sends preview image + saves file.

D5. Voice memo → transcription → action items

User dictates meeting notes → Dragon transcribes → extracts action items → renders as a checklist card with assignees.

D6. Photo + voice → combined context

User takes photo of their closet AND says "What should I wear today? It's cold outside" → AI considers both the wardrobe image AND weather.

D7. Text → audio story

"Tell my kid a bedtime story about a dragon" → AI writes story + generates it as a multi-voice audio narrative (not just TTS monotone).

D8. Shopping list → visual → share

"Make a grocery list from this recipe" → Dragon generates list → renders as shareable card with checkboxes → sends as image you can screenshot or share.

D9. Audio transcription → translation → audio

User records someone speaking French → Dragon transcribes → translates to English → reads back in English via TTS. Full audio-to-audio translation.

D10. Image carousel

"Show me 5 hairstyle ideas for short hair" → AI finds 5 images → Tab5 renders as swipeable carousel (not 5 separate messages).

D11. Before/after comparison

User sends a photo of messy room → AI suggests organization → Dragon renders a side-by-side "before/after" mockup.

D12. Video summary → key frames

User sends a long video → "What happens in this?" → Dragon extracts key frames + generates summary with timestamps.

D13. Daily briefing card

"Good morning" → Dragon renders a personalized morning card: weather, calendar, news headlines, reminders — visual + spoken summary.

D14. Shared conversation

"Share this answer with Sarah on WhatsApp" → Dragon formats the response (text + any images) and sends via TinkerClaw's WhatsApp channel.

D15. Iterative image refinement

"Generate a logo for my bakery" → AI creates image → "Make it more minimal" → new version → "Change the color to blue" → final version. Multi-turn image editing.

D16. Interactive quiz

"Quiz me on world capitals" → Dragon sends question cards with images of landmarks → user taps answer → Dragon shows result + next question.

D17. Photo journal entry

User takes a photo + says "I had an amazing hike today" → AI stores photo + text as a journal entry in memory → can recall later: "Show me my hike photos."

D18. Screenshare → help

Tab5 camera pointed at laptop screen → "Help me with this Excel formula" → AI reads the screen, understands context, gives specific help.

D19. Continuous monitoring

"Watch my sourdough and tell me when it's doubled" → Tab5 camera monitors → AI periodically checks → notifies with photo comparison when ready.

D20. Rich link sharing

User says a URL → AI fetches → renders Open Graph preview card (title, description, image, favicon) → tappable to "open" (Dragon renders full page screenshot).

D21. Annotation layer

User sends photo of a circuit board → AI responds with the SAME photo but annotated — circles, arrows, labels pointing to components.

D22. Voice → visual todo

"Add milk, eggs, and bread to my shopping list" → Dragon updates list → renders updated visual checklist → sends card.

D23. Multi-device media
