NOTES · 03IA principles

  • Telemetry on top. Mode · Dragon · WiFi · Bat · Uptime. The device shows its wiring. Taps: mode chip cycles; Dragon cell opens diagnostics.
  • Orb still wins. 168px with tick marks and a visible arc. The only "glowing" element — amber used sparingly elsewhere.
  • Agent as stack. Function-call syntax (inbox_scan(), draft_reply(aisha)) with timing + progress. Reads like `ps` + `systemctl status`.
  • Log as log. Two rows, each prefixed with a mono time and a colored tag. No cards at all.
  • Sharp corners. 2-4px radii throughout. The round orb is the exception — and that's what makes it pop.
▸ LVGL fit
All rectangles, no heavy shadow. Grid texture = optional, can skip on device. Tick marks = lv_canvas or 12 lv_lines at 30° angles (small win, optional).
