NOTES · 01IA principles

  • One hero. Orb at 160px — presence, not decoration. Arc ring is lv_arc (amber indicator, white track 4% opacity).
  • Kill the redundancy. Greeting → subtitle. Info cards → a single mono log. Agent summary is the only card above the fold.
  • Assertive mode. The status-bar chip shows rose-tinted CLAW · TINKERCLAW — tap cycles Local / Hybrid / Cloud / Claw. This is the one setting that rewires everything, so it gets air.
  • No filler. Timer becomes Memory (search past conversations). Two hardware affordances + one cognitive affordance.
  • Bottom chrome merged. Pill + nav share a single 128px zone but nav is visually quieter (uppercase mono, amber underline on active).
