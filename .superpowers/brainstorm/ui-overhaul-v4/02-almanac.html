NOTES · 02IA principles

  • Language, not cards. The agent speaks in first person: "I scanned your inbox…". Typography does the emotional work.
  • Orb as companion. 110px, off-center — part of the hero, not hoarding it. The big presence is the greeting.
  • Feed is a stream. Two compact rows with time + tag + title. No cards, no subtitles cluttering. Scroll-able underneath.
  • Three verbs at the edge. Camera / Files / Memory sit as thin pill-rows — affordances, not hero buttons.
  • Single bottom dock. Pill + nav share 124px but feel like one object thanks to the matching top rule.
▸ Trade-off
Fraunces italic is pretty but NOT on-device — on Tab5 we use Montserrat Bold 48 for the greeting (still distinctive via size + weight contrast, no serif).
▸ LVGL budget ~44 obj (smaller than A)
▸ Risk greeting-first IA depends on a named user — fallback "Good afternoon" if no name stored.
