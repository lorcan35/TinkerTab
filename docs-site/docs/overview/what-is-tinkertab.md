# What is TinkerTab?

:::info Coming soon
TinkerTab is the firmware that turns the M5Stack Tab5 into a voice-first home assistant. Tab5 is the face; the Dragon Q6A server is the brain. Together they handle voice, vision, video, skills, and onboard inference.
